# Web push notifications

# Commands

```shell
npx web-push generate-vapid-keys
```

# Environment Variables

Create a .env file with the following credentials:

- `PUBLIC_VAPID_KEY`
- `PRIVATE_VAPID_KEY`

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start
# Dev Server (Nodemon)
npm run dev
```
